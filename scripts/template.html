02 · 调优手段

Prompt Engineering

通过设计和优化提示词，使大模型更准确、更可控地产生所需输出。

SYS · 系统提示词

定义角色：身份、行为规则、约束边界

USR · 用户提示词

具体任务指令、Few-shot 示例、输出格式

2025-2026 新趋势

结构化输出 + Chain-of-Thought + Tree-of-Thought + ReAct · 提升效果，不改变模型参数

03 · 训练

Fine-tuning & LoRA

在已有模型基础上，用特定数据再训练，让模型更适合某个具体任务或场景。

Full Fine-tune

全量参数更新 · 显存占用大 · 效果最好 · 适合超大规模场景

LoRA · 低秩适配

只训练低秩矩阵 · 显存减少 70%+ · 2025 年已成企业微调主流

2026 应用：企业垂直领域微调（金融/医疗/法律）+ RLHF 对齐 · 相当于给通用模型注入"专业执照"

04 · 知识增强

RAG 检索增强生成

先从外部知识库检索相关信息，再结合这些信息一起生成回答。

→提升模型知识准确性与可信度
→实时更新知识，无需重新训练模型
→减少幻觉 · 可溯源 · 可审计

2026 新演进

GraphRAG（知识图谱）+ Hybrid Search（稀疏+稠密检索融合）· Milvus / Qdrant / Chroma

05 · Interface

Function Calling · MCP · A2A

Function Calling

大模型按约定格式输出调用指令，外部系统执行操作。实现"只会说话"到"会调用工具"的跨越。

MCP 协议

Anthropic 主推 · 标准化协议，AI 统一连接外部工具、数据源和服务。2025 年爆发，生态快速扩张。

A2A 协议

Agent-to-Agent 协议 · 不同厂商 Agent 之间的通信标准。解决多厂商 Agent 互联互通问题。

06 · 核心

Agent 智能体

能够基于目标进行 思考 → 行动 → 观察 循环，自主调用工具完成复杂任务的智能系统。

Prompt + LLM + Tools + Memory

本质是对人类行为的模拟 · 2026 年从"对话"进化为"代理执行"

07 · 协作

Multi-Agent 多智能体系统

多个分工协作的 Agent 共同完成任务，通过拆分任务与隔离上下文解决复杂问题。

Agent Teams（Anthropic）

Claude Opus 4.6 内置 · 多角色分工 · Agent 间原生通信 · 共享上下文

OpenAI Swarm

Handoffs 交接机制 · 多 Agent 编排实验框架 · 轻量级探索方案

注意事项：Token 消耗大 · 协作效率不稳定 · 系统复杂度高 · 需谨慎使用

08 · 上下文

Context Engineering

Agent 运行中的一切相关信息都是上下文：对话历史、用户输入、背景知识、工具结果……

→高质量筛选上下文（信息密度 > 信息量）
→有效压缩信息量（摘要 / 重要性排序 / 滑动窗口）
→最大化模型推理能力（上下文窗口优化 / RAG 协同）

2026 趋势

128K+ 上下文窗口普及，但"压缩"能力成为核心竞争力

09 · 能力封装

Agent Skill 技能封装

将一整套 Agent 能力（Prompt、工具脚本，知识文件）封装为可复用模块，按需激活。

本质

Agent Skill ≈ 子 Agent ≈ SOP 的沉淀和复用

"离职的同事终将化作温暖的 Skill"

SkillHub 生态 · 2026 年收录 1.3 万+ Skills · 覆盖各行业场景，开箱即用

10 · 框架

OpenClaw 开源 Agent 框架

开源、高可扩展的 AI Agent 框架，基于 TypeScript 开发，用于构建可自定义的私人 AI 助手。

✓多平台接入（飞书 · Telegram · Discord · 钉钉 · 企业微信）
✓Skill 生态 SkillHub（1.3 万+ Skills，腾讯 CDN 加速）
✓MCP 原生支持 + 记忆系统 + Cron 定时任务 + Heartbeat
✓支持 DeepSeek · 豆包 · GPT · Claude 等多模型
11 · 工程实践

Harness Engineering

构建受控环境，让 Agent 在约束下高效可靠地完成长周期复杂任务。从"优化模型"转向"优化模型运行环境"。

约束机制

Guardrails 护栏 · 权限边界 · 不可逾越红线 · 操作审计

反馈回路

自验证循环 · 纠错机制 · 持续改进 · 错误不重复发生
