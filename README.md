# ppt-generator-biz

> 商业风格 16:9 HTML 演示文稿生成器 · 白底红字配色 · 开箱即用

**Live Demo**: https://9c6p2110cg9b.space.minimaxi.com

---

## Features

- **16:9 横屏强制适配** — 无论设备，始终保持标准演示比例
- **白底红字商务配色** — 纯白背景 + 深黑文字 + 红色强调
- **字体足够大** — 实测在投影/大屏上清晰可见
- **全居中排版** — 标题、正文、卡片全部居中
- **三列改上下堆叠** — 内容多时纵向展开，不再被压缩
- **纯浏览器实现** — 无需任何依赖，单 HTML 文件走天下
