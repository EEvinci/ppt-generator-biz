```

   ↓
```

## 内容规模规范

### 页类型配置

| 类型 | 数量 | 说明 |
|------|------|------|
| cover | 1 | 封面：标题 + 副标题 + 红色分割线 |
| overview | 1 | 目录：三列网格，4-6 个章节 |
| section | 8-12 | 内容页：每页 5-6 个卡片 |
| end | 1 | 结尾：金句收尾，白底极简 |

**目标：16-20 页 / 次**

### 章节规划模板（根据主题调整）

```
章节1：概念定义 — 是什么，为什么重要
章节2：核心原理 — 底层逻辑，为什么这样运作
章节3：核心能力 — 优势详细展开，含真实案例
章节4：应用场景 — 典型场景 + 效果数据
章节5：数据佐证 — 权威数据量化说明
章节6：局限与行动 — 客观分析 + 具体建议
```

### 内容充实度要求（强制）

每个卡片正文必须：
- **5-7句话**（不是2-3句）
- 包含**具体案例 / 真实数字 / 场景描述**
- 包含**观点解释**的逻辑，不只是结论
- 避免空洞结论，要细节支撑

### 示例对比

```
❌ 不合格：AI 提升效率，效果显著。（太短，无细节）

✅ 合格：AI 将任务完成时间从「小时级」压缩到「分钟级」，
同等质量下速度提升 5-20 倍。比如一份市场调研报告，人工需要
2-3 天，AI 辅助下 2-3 小时即可交付初稿，效率提升 10 倍以上，
这是最直接的效率提升。
```

## 版式标准（唯一标准版）

**纵向卡片列表版式：**
- 5-6 个卡片纵向排列（单列）
- 每个卡片：标题（粗体居中）+ 正文（居中，5-7句）
- 白色背景，红色标签徽章，红色分割线
- **所有文字居中**

### 字号体系

| 元素 | 字号 | 居中 |
|------|------|------|
| 封面标题 | 3.2-3.4em | ✅ |
| 页面标题 | 2.0-2.2em | ✅ |
| 标签徽章 | 0.78em | — |
| 卡片标题 | 1.05em | ✅ 必须 |
| 卡片正文 | 0.95em | ✅ 必须 |

### 配色方案

| 用途 | 颜色 |
|------|------|
| 背景 | `#fff` 纯白 |
| 标题文字 | `#1a1a2e` 近黑 |
| 强调/红色 | `#c0392b` |
| 绿色 | `#27ae60` |
| 蓝色 | `#2980b9` |
| 黄色 | `#c9970c` |
| 紫色 | `#8e44ad` |
| 次要文字 | `#888` |

## slide type 速查

| type | 用途 | 关键参数 |
|------|------|---------|
| `cover` | 封面 | title + subtitle，**不要封面标签** |
| `overview` | 目录 | ov-grid 三列网格，6-10 个章节 |
| `section` | 内容页 | 5-6 个 cards，所有文字居中 |
| `end` | 结尾 | 简洁有力，白底，不要装饰 |

---

## 封面设计规范

封面**只保留**：
- 主标题（大字，黑）
- 副标题（小字，灰）
- 红色分割线

**禁止出现**：
- "封面" / "Cover" 标签文字
- 翻页提示语

---

## 目录页设计

使用 **三列网格** 布局，序号格式 01-10：

```html
<div class="ov-grid">
  <div class="ov-item">
    <div class="ov-num">01</div>
    <div class="ov-text">章节名</div>
  </div>
</div>
```

## 内容页版式

```html
<div class="s">
  <div class="sec-header">
    <div class="tag-badge">01 · 章节名</div>
    <h2 class="page-title">页面标题</h2>
    <div class="bar-line"></div>
  </div>
  <div class="card-list">
    <div class="card">
      <div class="card-title">卡片标题</div>
      <div class="card-body">正文内容，5-7句，真实案例，数字支撑...</div>
    </div>
    <!-- 重复 5-6 个 card -->
  </div>
</div>
```

## 铁律（最高优先级）

> ⚠️ **CSS 禁止使用 Python `%` 格式化**
> `width: 100%;` 中的 `%` 与 Python `%` 格式化冲突，导致浏览器 CSS 解析失败，页面全白。
> **CSS 只能写纯字符串拼接（`+`），绝对不能用 `%` 格式化。**

| 坑 | 解法 |
|----|------|
| CSS `%%` 冲突 | CSS 用纯字符串，`+` 拼接 |
| 首屏空白 | 首屏 `<div class="s active">` 硬编码 |
| 内容太少 | 每次目标 20 页，每页 5-6 卡片 |
| 封面过度设计 | 只留标题+副标题+分割线 |
| 目录无结构 | ov-grid 三列网格 + ov-num 序号 |
| 文字不居中 | `.card-title` 和 `.card-body` 显式加 `text-align:center` |
